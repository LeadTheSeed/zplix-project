# مدير المهام متعدد اللغات | Multilingual Task Manager

## نظرة عامة | Overview

هذا مشروع تطبيق ويب متكامل (Full Stack) بدعم اللغتين العربية والإنجليزية. يتيح التطبيق للمستخدمين إنشاء حسابات وإدارة مهامهم اليومية.

This is a Full Stack web application with support for Arabic and English languages. It allows users to create accounts and manage their daily tasks.

## المميزات | Features

- ✅ دعم اللغتين العربية والإنجليزية | Support for Arabic and English languages
- ✅ إنشاء حساب وتسجيل الدخول | User registration and login
- ✅ إضافة وتعديل وحذف المهام | Add, edit, and delete tasks
- ✅ تحديد أولوية وتاريخ استحقاق للمهام | Set priority and due date for tasks
- ✅ تصميم متجاوب مع جميع الأجهزة | Responsive design for all devices

## التقنيات المستخدمة | Technologies Used

### الواجهة الأمامية | Frontend
- React.js
- TypeScript
- Material UI
- i18next (للترجمة | for translation)
- React Router
- Axios

### الخادم الخلفي | Backend
- Node.js
- Express.js
- MongoDB
- JSON Web Token (للتوثيق | for authentication)
- Mongoose

## كيفية التشغيل | How to Run

### متطلبات النظام | Requirements
- Node.js (v14+)
- npm or yarn
- MongoDB (محلي أو اتصال عن بعد | local or remote connection)

### تشغيل الخادم الخلفي | Running the Backend
```bash
cd backend
npm install
npm run dev
```

### تشغيل الواجهة الأمامية | Running the Frontend
```bash
cd frontend
npm install
npm start
```

## الاستضافة | Deployment
- الواجهة الأمامية مستضافة على | Frontend hosted on: [Netlify/Vercel]
- الخادم الخلفي مستضاف على | Backend hosted on: [Render/Heroku]
- قاعدة البيانات مستضافة على | Database hosted on: [MongoDB Atlas]
