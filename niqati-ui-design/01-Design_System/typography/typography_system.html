مقاسات الخطوط | Font Sizes

المتغير الحجم مثال الاستخدام
--text-xs 0.75rem (12px) هذا نص بحجم xs النص التوضيحي الصغير، التنبيهات
--text-sm 0.875rem (14px) هذا نص بحجم sm التعليقات، البيانات الثانوية
--text-base 1rem (16px) هذا نص بحجم base نص الفقرات الرئيسي
--text-lg 1.125rem (18px) هذا نص بحجم lg النصوص البارزة، الفقرات المهمة
--text-xl 1.25rem (20px) هذا نص بحجم xl عناوين البطاقات الصغيرة
--text-2xl 1.5rem (24px) هذا نص بحجم 2xl عناوين الأقسام
--text-3xl 1.875rem (30px) هذا نص بحجم 3xl العناوين الفرعية للصفحات
--text-4xl 2.25rem (36px) هذا نص بحجم 4xl العناوين الرئيسية
--text-5xl 3rem (48px) هذا نص 5xl عناوين الصفحات الرئيسية

تباعد السطور | Line Heights

تباعد السطور الافتراضي للنص العادي: 1.6

منصة نقاطي هي منصة تداول USDT بواجهة نقاط رقمية سهلة الاستخدام. تم تصميمها خصيصاً لمستخدمي الشرق الأوسط وشمال إفريقيا مع مراعاة التوافق مع الشريعة الإسلامية. هذا مثال على النص مع تباعد السطور الافتراضي.

تباعد السطور للعناوين: 1.2

العناوين تحتاج إلى تباعد أقل بين السطور لتبدو أكثر تماسكاً وقوة في التصميم.

تباعد السطور للنصوص الصغيرة: 1.5

النصوص الصغيرة مثل التعليقات والملاحظات تحتاج إلى تباعد سطور مناسب لتحسين القراءة. هذا مثال على نص توضيحي صغير مع تباعد سطور 1.5 لتحسين القراءة وسهولة الوصول.

توجيهات للاستخدام | Usage Guidelines

العناوين الرئيسية | Main Headings

عنوان رئيسي

استخدم حجم --text-4xl مع وزن Bold (700) أو Black (900) للعناوين الرئيسية.

العناوين الفرعية | Subheadings

عنوان فرعي

استخدم حجم --text-2xl مع وزن Semibold (600) للعناوين الفرعية.

نص الفقرات | Body Text

هذا مثال لنص فقرة. يجب أن يكون النص واضحاً ومقروءاً على جميع أحجام الشاشات.

استخدم حجم --text-base مع وزن Regular (400) لنص الفقرات.

النص التوضيحي | Caption Text

هذا مثال لنص توضيحي صغير يستخدم للمعلومات الإضافية والملاحظات.

استخدم حجم --text-xs أو --text-sm للنصوص التوضيحية.

استخدام متغيرات CSS | CSS Variables

:root {
    /* Font Families */
    --font-ar: 'Cairo', sans-serif;
    --font-en: 'Inter', sans-serif;
    
    /* Font Sizes */
    --text-xs: 0.75rem;     /* 12px */
    --text-sm: 0.875rem;    /* 14px */
    --text-base: 1rem;      /* 16px */
    --text-lg: 1.125rem;    /* 18px */
    --text-xl: 1.25rem;     /* 20px */
    --text-2xl: 1.5rem;     /* 24px */
    --text-3xl: 1.875rem;   /* 30px */
    --text-4xl: 2.25rem;    /* 36px */
    --text-5xl: 3rem;       /* 48px */
}

استخدم متغيرات CSS للحفاظ على اتساق الخطوط في جميع أنحاء التطبيق. مثال:

.heading-main {
    font-family: var(--font-ar);
    font-size: var(--text-4xl);
    font-weight: 700;
}

.button-label {
    font-size: var(--text-sm);
    font-weight: 600;
}

/* English version */
html[lang="en"] .heading-main {
    font-family: var(--font-en);
}
