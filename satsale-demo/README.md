# نموذج SatSale لمشروع Zplix

هذا نموذج بسيط لتكامل SatSale مع مشروع Zplix. يوضح هذا النموذج كيفية إعداد صفحة دفع بالبيتكوين والتي يمكن تضمينها في أي موقع ويب.

## الميزات المتاحة في هذا النموذج

- صفحة دفع أساسية بالبيتكوين
- إعداد مبسط باستخدام xpub (دون الحاجة إلى تشغيل عقدة بيتكوين كاملة)
- واجهة مستخدم نظيفة وسهلة الاستخدام
- إمكانية تضمين زر التبرع في أي موقع

## كيفية التشغيل

1. قم بتثبيت المتطلبات:
```
pip3 install -r requirements.txt
```

2. قم بتشغيل الخادم المحلي:
```
gunicorn -w 1 -b 0.0.0.0:8000 app:app
```

3. الوصول إلى التطبيق على:
```
http://127.0.0.1:8000
```

## تضمين زر الدفع في موقعك

```html
<iframe
  src="http://127.0.0.1:8000/"
  style="margin: 0 auto;display:block;width:420px;height:460px;border:none;overflow:hidden;"
  scrolling="no"
></iframe>
```

## المميزات التي يمكن إضافتها مستقبلاً

- دعم شبكة البرق (Lightning Network)
- تكامل مع تطبيقات التجارة الإلكترونية
- تتبع المدفوعات والإشعارات
