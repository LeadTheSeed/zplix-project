استخدام متغيرات CSS | CSS Variables

:root {
    /* Primary */
    --primary-500: #7B3FF2;
    --primary-400: #946DF6;
    --primary-600: #6232D2;
    
    /* Secondary */
    --secondary-500: #00D4AA;
    
    /* Semantic */
    --success-500: #10B981;
    --error-500: #EF4444;
    --warning-500: #F59E0B;
    
    /* Backgrounds */
    --bg-dark: #0F0F14;
    --bg-card-dark: #1A1A23;
}

استخدم متغيرات CSS للحفاظ على اتساق الألوان في جميع أنحاء التطبيق. مثال:

.button-primary {
    background-color: var(--primary-500);
    color: white;
}

.button-primary:hover {
    background-color: var(--primary-400);
}

.button-primary:active {
    background-color: var(--primary-600);
}
