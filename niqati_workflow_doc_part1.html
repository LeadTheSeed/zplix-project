Project Overview

Niqati is a comprehensive digital points marketplace platform designed to serve the entire MENA region. This document outlines the complete end-to-end workflow and technical specifications for transforming the MVP into a full-scale production system with advanced features including multi-currency support, cryptocurrency integration, user authentication systems, and digital wallet functionality.

Key Objectives

  • Develop a robust, scalable platform supporting all MENA countries and currencies
  • Implement comprehensive user authentication with email and OTP verification
  • Integrate multiple payment gateways with localized payment methods
  • Establish USDT cryptocurrency settlement system for merchants
  • Create digital wallet functionality with balance management
  • Deploy advanced security measures and fraud prevention
  • Support multilingual interfaces (Arabic/English) with RTL/LTR layouts
Target Market Coverage: UAE, Saudi Arabia, Egypt, Jordan, Kuwait, Qatar, Bahrain, Oman, Morocco, Tunisia, Algeria, Lebanon, Iraq, and other MENA countries.

نظرة عامة على المشروع

نقاطي هي منصة شاملة لسوق النقاط الرقمية مصممة لخدمة منطقة الشرق الأوسط وشمال أفريقيا بأكملها. توضح هذه الوثيقة سير العمل الكامل من البداية إلى النهاية والمواصفات الفنية لتحويل النموذج الأولي إلى نظام إنتاج كامل النطاق مع ميزات متقدمة تشمل دعم العملات المتعددة، وتكامل العملات المشفرة، وأنظمة مصادقة المستخدم، ووظائف المحفظة الرقمية.

الأهداف الرئيسية

  • تطوير منصة قوية وقابلة للتوسع تدعم جميع دول وعملات منطقة الشرق الأوسط وشمال أفريقيا
  • تنفيذ مصادقة شاملة للمستخدمين مع التحقق عبر البريد الإلكتروني ورمز OTP
  • دمج بوابات دفع متعددة مع طرق دفع محلية
  • إنشاء نظام تسوية بالعملة المشفرة USDT للتجار
  • إنشاء وظائف محفظة رقمية مع إدارة الرصيد
  • نشر تدابير أمنية متقدمة ومنع الاحتيال
  • دعم واجهات متعددة اللغات (العربية/الإنجليزية) مع تخطيطات RTL/LTR
تغطية السوق المستهدفة: الإمارات، السعودية، مصر، الأردن، الكويت، قطر، البحرين، عمان، المغرب، تونس، الجزائر، لبنان، العراق، وغيرها من دول الشرق الأوسط وشمال أفريقيا.
