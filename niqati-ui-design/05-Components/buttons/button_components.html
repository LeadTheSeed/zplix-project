مكونات الأزرار لمنصة نقاطي

دليل شامل لتصميم وأنماط الأزرار المستخدمة في منصة نقاطي للتداول

الأزرار الأساسية | Primary Buttons

الأزرار الملونة | Colored Buttons

الأزرار الأساسية بألوان مختلفة تستخدم للإجراءات المختلفة في التطبيق.

<button class="btn btn-primary">زر رئيسي</button> <button class="btn btn-secondary">زر ثانوي</button> <button class="btn btn-success">زر النجاح</button> <button class="btn btn-warning">زر التحذير</button> <button class="btn btn-error">زر الخطأ</button>

أزرار التدرج | Gradient Buttons

أزرار مميزة بخلفيات متدرجة للإجراءات الرئيسية والمميزة.

<button class="btn btn-gradient-primary">زر رئيسي متدرج</button> <button class="btn btn-gradient-success">زر النجاح المتدرج</button> <button class="btn btn-gradient-error">زر الخطأ المتدرج</button> <button class="btn btn-gradient-premium">زر مميز</button>

أزرار الإطار والشفافية | Outline & Ghost Buttons

أزرار الإطار | Outline Buttons

أزرار بإطار خارجي بدون خلفية، تستخدم للإجراءات الثانوية.

<button class="btn btn-outline">زر إطار</button>

أزرار شفافة | Ghost Buttons

أزرار شفافة بدون خلفية أو إطار، تستخدم للروابط والإجراءات الخفيفة.

<button class="btn btn-ghost">زر شفاف</button>

أحجام الأزرار | Button Sizes

أحجام مختلفة | Different Sizes

أزرار بأحجام متعددة للاستخدامات المختلفة.

<button class="btn btn-primary btn-lg">زر كبير</button> <button class="btn btn-primary">زر متوسط</button> <button class="btn btn-primary btn-sm">زر صغير</button>

أزرار الأيقونات | Icon Buttons

أزرار مربعة تحتوي على أيقونات فقط.

<button class="btn btn-primary btn-icon">+</button> <button class="btn btn-secondary btn-icon">✓</button> <button class="btn btn-error btn-icon">✕</button> <button class="btn btn-outline btn-icon">↻</button>

أشكال الأزرار | Button Shapes

أزرار مستديرة | Rounded Buttons

أزرار بحواف مستديرة بشكل كامل.

<button class="btn btn-primary btn-rounded">زر مستدير</button> <button class="btn btn-outline btn-rounded">زر إطار مستدير</button> <button class="btn btn-gradient-primary btn-rounded">زر متدرج مستدير</button>

حالات الأزرار | Button States

زر معطل | Disabled Button

حالة الزر عندما يكون غير متاح للضغط.

<button class="btn btn-primary btn-disabled">زر معطل</button> <button class="btn btn-outline btn-disabled">زر إطار معطل</button>

أزرار مع أيقونات | Buttons with Icons

أزرار تحتوي على نص وأيقونة.

<button class="btn btn-primary">✓ تأكيد</button> <button class="btn btn-error">✕ إلغاء</button> <button class="btn btn-success">⬇️ تنزيل</button> <button class="btn btn-warning">⚠️ تحذير</button>
